Restaurant Order Analysis

Objectives

This analysis will look into the trend of orders of a given restaurant.

Key Findings

  • The maximum orders per day is 85 orders and it occured on Feb 1, 2023.
  • The minimum orders per day occured on Feb 22, 2023 with 35 orders.
  • With outliers, Wednesday has the most variablity when it comes to restaurant orders.
  • Without outliers, Sunday has the most variablity when it comes to restaurant orders.
  • The peak hours of the restaurant are during 12:00 (12:00 PM) and 17:00 (5:00 PM).

Approach

  1. Obtained the data from Maven Analytics - Data Playground.
  2. Used the Python Library Plotly to create a dashboard.
  3. Used a line plot to plot the number of orders per day.
  4. A box and whisper plot was used to plot the statistics of order per day of the week.
  5. A bar plot was used to plot the number of average orders per hour.
