Lego Set Analysis

Objectives

This analysis focuses on exploring the following for Lego Sets that were introduced from 2015 to 2022 and in the Normal Category:

  • Distribution of Theme Group per year
  • Top 5 Themes according to the number of sets
  • Top 5 Themes according to the number of pieces per set
  • Relationship between the numer of pieces per set and the their corresponding retail price

Key Findings

  • “Licensed” is the top theme group from 2015 to 2022.
  • Star Wars was the top theme according to the number of sets until the year 2018 when it was overtaken by BrickHeads.
  • City was consistently in the top 3 and was able to reach the top 1 in 2022.
  • Star Wars was consistently in the top 5 according to pieces
  • The Icons theme was in 1st place in the years 2021 and 2022.
  • The overall relationship of the two variables is positive where a higher number of pieces would likely correspond to a more expensive Lego Set.

Approach

  1. Obtained the data from Maven Analytics - Data Playground.
  2. Filtered the data set in Python to suit the analysis.
  3. Used the Python Library Plotly to create an interactive dashboard.
